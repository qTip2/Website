Custom build

If you're a bit more tech savvy and/or want control voer exactly what plugins are included in your qTip build, you can custom build our own qTip script using the GitHub repo and Apache Ant or *nix make.

First off, you'll need to get acquainted with GitHub using their Help Guide. Once you've installed Git it's time to clone the qTip repository located here. You can do this by running this on the command line:

highlightCode('sh', "git clone git://github.com/Craga89/qTip.git"); ?>

This will copy all the files from the GitHub repository to the current folder you ran the command in. Once you have the files, you'll need to decide what plugins you wish to include in your qTip build. Here's a quick list of the ones included by default:

highlightCode('sh', 'PLUGINS="src/ajax.js src/tips.js src/imagemap.js src/modal.js src/bgiframe.js"'); ?>
