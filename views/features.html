Find out what qTip has to offer

Features

  • Cross browser
  • Degradable
  • Tips
  • Easy Positioning
  • Stylish
  • Stacking
  • Custom Effects
  • AJAX
  • Scripting API

  • All modern browsers supported

    qTip supports all the major browsers out there (even Internet Explorer!) including:
    6.0+
    2.0+
    9.0+
    3.0+
    1.0+
    3.5+
     

    Note: We're constantly resolving browser-dependant issues so if you find a bug, report it!

  • JavaScript disabled? No problem!

    All of qTip's HTML is generated on-the-fly so those without JavaScript enabled see no ugly leftovers, making qTip completely degradable!

  • Speech-bubble effects

  • Forget your x and y's

    Throw out your rulers boys and girls, because positioning tooltips in qTip is as easy as picking two corners thanks to jQuery UI's Position utility.   • Stylish to the max

    qTip comes pre-packed with 7 elegant tooltip styles for use in your applications!

  • Keep the focus

    When you've got a lot of tooltips on a page, especially ones which appear pretty close together, you can run into problems with overlapping. Not a problem with qTip! Tooltips automatically focus when hovered over, and even preserve their stacking order on the page.

    One tooltip Two tooltips Three tooltips Four tooltips And five!

    Hover over the tooltips above for a demonstration. Notice how the stacking order is preserved.

  • Click the tooltips! Create some funky effects

    By default, qTip comes with only a single fade effect for showing/hiding your tooltips. But support for custom effect functions means that you've got an unlimited amount of effects at your fingertips!

    Note: Some of the custom effects here use the jQuery UI library, but it is not required if you wish to make your own animations!

  • Dynamic content at your fingertips

    Thanks to jQuery's AJAX support, content can be loaded dynamically even after the page is loaded! Infact, the tooltip below loads new content everytime its clicked... try it and see.
     

    Note: Content is hosted locally but is taken from the almighty Wikipedia.

     

  • Unleash your inner developer

    qTip comes with a fully fledged scripting API allowing you to easily extend the libraries functionality through a variety of optional callback methods and library attributes.

    It's even possible to write your own qTip plugins (plugins for a plugin you say!?) using this API, so developers... get coding!

    For full details on the Scripting API checkout the documentation!
